Project description

Quiz Plus is developed for your institutions like Schools,Colleges,Coaching centers,Companies or any other institutes who want to take online or offline (in case of LAN) Quizes, Exams, Tests, Competitions, Challenges or Surveys.

Quiz Plus is a single page web application developed by Vue.js with .NET core 5 API and you have flexibility to choose Sql server, Mysql, Sqlite, PostgreSql or Oracle 12c+ as database provider.

 

 

 

Project Requirement

  • Visual Studio Code (preffered) / Visual Studio 2019.

  • Database provider software (SQL Server / MySql / Sqlite / PostgreSql / Oracle).

Set up QuizPlusApi

  • Open QuizPlusApi project on Visual studio code.On appsettings.json file you will find five connection strings for five different database connectivity. Replace the connection string by your one.

"ConnectionStrings": {

    "ApiConnStringMssql": "data source=Sangib-PC;initial catalog=QuizPlusDB;
     persist security info=True;Integrated Security=SSPI;",

    "ApiConnStringMysql": "server=localhost;port=3306;database=QuizPlusDB;user=root;",

    "ApiConnStringOracle": "Data Source=(DESCRIPTION=(ADDRESS_LIST=
    (ADDRESS=(PROTOCOL=TCP)(HOST=localhost)(PORT=1521)))  
    (CONNECT_DATA=(SERVER=DEDICATED)(SERVICE_NAME=orclpdb)));User Id=root;Password=root;",

    "ApiConnStringSqlite": "Data Source=QuizPlusDB.db;",

    "ApiConnStringPostgreSql": "Server=127.0.0.1;Port=5432;Database=QuizPlusDB;
    User Id=postgres;Password=postgres;"

  }

 

  • Then open Startup.cs file, On ConfigureServices method you will see five AppDbContextPool for five Connection Strings that already have on appsettings.json file. Un-Comment your required one and keep other four as commented.Have a look to the below code.I only un-comment the Mysql portion because i used Mysql DB here.

  • Now Open terminal on VS Code and write below commands     

               ->dotnet ef migrations add <MigrationName>

               ->dotnet ef database update

               ->dotnet run

  • After successful build a folder named Migrations should create on project structure and a database named   QuizPlusDB  should create with seed data to your MySQL server (if you use MSSQL then you should check to Sql Server and so on for other databases). After executing "dotnet run" command you should see that your API listening at http://localhost:5001/.Navigate to the link to see the swagger Api documentation UI.

  • You can check the Swagger API documentation UI by navigating http://localhost:5001/.First go to the /api/User/GetLoginInfo/{email}/{password} method from documentation page then enter email and password as i mentioned eairlier, a token will be generated here. Use this token to authorize the api by clicking the Authorize button at the top right and header value format example: "Bearer TokenText". Then you will able to check all the methods as expected.

  • You can change token expire time from UserController GenerateJwtToken method.Currently it is set to 360 minutes.

        string GenerateJwtToken(UserInfo userInfo)
        {
            var securityKey=new SymmetricSecurityKey(Encoding.UTF8.GetBytes
                           (_config["Jwt:SecretKey"]));
            var credentials = new SigningCredentials(securityKey, 
                              SecurityAlgorithms.HmacSha256);
            var claims = new[]
            {
                new Claim(JwtRegisteredClaimNames.Sub, userInfo.UserId.ToString()),
                new Claim("fullName", userInfo.FullName.ToString()),
                new Claim("role",userInfo.RoleName),
                new Claim(JwtRegisteredClaimNames.Jti, Guid.NewGuid().ToString()),
            };
            var token = new JwtSecurityToken(
                issuer: _config["Jwt:Issuer"],
                audience: _config["Jwt:Audience"],
                claims: claims,
                expires: DateTime.Now.AddMinutes(360),
                signingCredentials: credentials
            );
            return new JwtSecurityTokenHandler().WriteToken(token);
        }

 

  • After first migration all the project specific data will insert to your underlying database through your Models->AppDBContext.cs class OnModelCreating Seed method.

        protected override void OnModelCreating(ModelBuilder modelBuilder)   
        {   
            modelBuilder.Seed();//use this for Sql server,Mysql,Sqlite,Oracle and PostgreSql
        }

 

  • This project used database driven role based authorization. Through migration data you will get two roles, Admin and Student. You can add more role from Roles menu after running the Client project. These role names are used in Authorize attribute on Api project to authorize that particular method.

  • In appSettings.json file there is a MailSettings object. Update your Email & Password from here.

 

 

Set up QuizPlusClient

  • Open QuizPlusClient on Visual Studio Code.On terminal first write "npm install", it will add node modules to project then write "npm run serve".

  • Now open config.js file, you should see a hostname here which is set to http://localhost:5001. Actually QuizPlusApi project is running here.Change this url in case of actual hosting.

  • After doing above settings your client project will be running at http://localhost:8080

Dashboard

In Admin Dashboard, application status will show by different charts & data. Charts are based on different data like

  • Quiz Participation (Date wise)

  • Participation (Quiz Topic wise)

  • Browsing history (Month wise)

  • Browsing history (Depends on browser)

  • Browsing history (Operating system wise)

  • Login (Date wise)

 

In Student Dashboard, all above charts are available only for that logged in student and on top of all charts there will be a list of available quiz/exam/survey.

Menus

Here you find a list of all menus of this App. To add a new menu you have to fill below fields

  1. Menu Title (No duplicate Menu Title is accepted)

  2. URL (No duplicate URL is accepted)

  3. Order No. (No duplicate order no. is accepted)

  4. Icon (Only material design icon is allowed, browse this link to find more icons https://fonts.google.com/icons?selected=Material+Icons)

Roles

You will find a list of two roles (Admin & Student) initially. These two roles are restricted to modify and delete because these two roles are used for many configuration issue.

Anyway you can add new roles and can modify or delete them as per your need.  

Users

Initially you will find two users here for testing purpose. You can modify those and can add new users from here.

Quiz/Exam/Survey

It is the starting point of a new Quiz/Exam/Survey. You can create new Exam or quiz or survey by filling below fields,

  1. Title (Title of your Quiz/Exam/Survey, No duplicate is accepted)

  2. Mark Option (Two options available i.e Question wise & No marks(survey), actually here you will define whether it is an marks based exam or an survey without marks).

  3. Multiple Attempt ( Check this only If you allow your students to take a quiz/exam/survey multiple times)

  4. Multiple Answer ( Check this if you want to set multiple answers for one question)

  5. Multiple Attempt ( Check this only If you allow your students to select multiple answer for one question)

  6. Marks (only visible for Question wise Mark Option)

  7. Required Time (only visible for Question wise Mark Option and must be set in minutes)

  8. Pass Marks (only visible for Question wise Mark Option and one student have to pass for getting a certificate)

  9. Schedule Start Date ( in case of schedule a exam/quiz)

  10. Schedule End Date ( in case of schedule a exam/quiz)

  11. Certificate Template ( select one if you want to give certificate)

  12. Correct Answer display to students ( Check this if you allow correct answers to student result sheet)

  13. Participant Option ( All registered students or you may set a comma seperated email list like abc@gmail.com, xyz@hotmail.com,def@yahoo.com)

Questions

List of Quiz/Exam/Survey will find here and you can add questions for each Quiz by filling below fields

  1. Question (No duplicate is accepted for a particular quiz)

  2. Code Snippet ( Check this only if you want set pre-formatted questions like Programming language code or anything you want keep source format)

  3. Question Video ( Only youtube embed video is allowed) (Optional)

  4. Question Image (Optional)

  5. Option-A

  6. Option-B

  7. Option-C (Optional)

  8. Option-D (Optional)

  9. Option-E (Optional)

  10. Correct Answer

  11. Answer Explanation (Optional)

Reports

All your students exam records will find here. You can filter those and can download those on PDF & Excel .

App Settings

Settings have two categories, General Settings & Color Settings.

In General Settings there will be some initial values and you can change them as per your preference.

In Color Settings, Initially all are empty and you can play with App Bar, Footer and Body Colors.
